# Deployment Evidence Folder

Save backend and frontend deployment proof images in this folder.

## Required Files
- backend-render-overview.png
- backend-live-url-response.png
- frontend-vercel-overview.png
- frontend-live-url-view.png

## Mapping
- backend-render-overview.png: Render service page showing deployment status
- backend-live-url-response.png: Browser or Postman response from backend live URL
- frontend-vercel-overview.png: Vercel deployment overview page
- frontend-live-url-view.png: Running frontend application at live URL
